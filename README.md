# PAA-Tugas03-2411600469-Nurliah Awaliah

## Requirement
- Java 17

## How to Run

- With gradle

```bash
./gradlew runMain
./gradlew runSegitiga
```

- Without Gradle
```bash
./gradlew build # one time process
java -cp build/classes/java/main Main
java -cp build/classes/java/main Segitiga
```
